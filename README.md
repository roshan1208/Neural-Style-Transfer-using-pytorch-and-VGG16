### Create Conda Environment
* Create Conda Virtual Environment for better experience
  ```sh
  conda create --name NST
  ```
  Then
  ```sh
  conda activate NST
  ```
### Now Installation
* pytorch
  ```sh
  pip install torch
  ```
* Numpy
  ```sh
  pip install numpy
  ```
* Matplotlib.pyplot
  ```sh
  pip install matplotlib.pyplot
  ```
* PIL
```sh
pip install pil
```
